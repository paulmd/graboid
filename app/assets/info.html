What does this app do?
Graboid records a copy of the data on a MIFARE contactless smart card and stores it on the phone. Using the data stored on the phone, it is then possible to restore the same card to it's previous state.

What this app doesn't do?
You can not use Graboid to make a copy of one card and transfer it to another card.

What about keys?
Before using Graboid to record and restore a card, the app has to be configured to use a set of MIFARE keys.

Keys are imported from key files with the *.gkeys extension. Search for key files on the Internet or create your own.

The key files are plain text files containing the two keys (A and B), separated by a space. One sectors key pair per line.

Warning
Please use Graboid responsibly. Using a restored MIFARE card might be illegal in certain circumstances.
